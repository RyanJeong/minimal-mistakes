* C ver.
```bash
set number
syntax on
set softtabstop=4
set tabstop=4
set shiftwidth=4
set expandtab
```

* C++ ver.
```bash
set number
syntax on
set softtabstop=2
set tabstop=2
set shiftwidth=2
set expandtab
```
